MOBILE APPS  F.A.Q

(Frequently Ask Question)

Bahasa Indonesia
  1. Masalah Login atau tidak dapat login
    • Periksa nama pengguna / kata sandi / kode perusahaan apakah sudah benar?
    • Periksa apakah aplikasi GreenHCM sudah ke versi terbaru? jika belum menggunakan versi terbaru, anda dapat memperbaruinya di Google Playstore disini & IOS disini
    • Periksa GPS dan sinyal seluler Aktif dan oke?
    • Periksa apakah pengaturan izin Telepon untuk penyimpanan / kamera / lokasi sudah dihapus untuk aplikasi GHCM?
    • Periksa apakah sumber daya Telepon (Penyimpanan & Memori) sudah cukup. Jika tidak, umumnya dapat diatasi dengan membersihkan ruang penyimpanan dengan menghapus file yang tidak digunakan dan membersihkan Memori Telepon (RAM).
  2. Aplikasi GreenHCM tidak mau terinstall.
    • Pastikan bahwa O/S selular anda adalah Android v.5 (Lollypop) keatas.
    • Periksa sumber daya telepon anda. Jika penuh, bebaskan ruang penyimpanan dengan menghapus file-file atau aplikasi yang sudah tidak diperlukan, juga membebaskan rekaman memori telepon.
  3. Anda lupa username / password / kode company?
    • Untuk Password, anda bisa meresetnya dari link 'Forgot Password' di mobile app atau Web portal. Untuk Kode Perusahaan atau Username, anda bisa cek Email atau SMS verifikasi pemberitahuan yang terakhir dikirimkan saat registrasi akun awal.
    • Jika masih belum bisa, anda bisa meminta bantuan Admin PIC sistem GreenHCM di perusahaan anda untuk mencarikannya di sistem.
    • Sebagai bantuan akhir, anda dapat menghubungi layanan pelanggan kami untuk membantu anda.
  4. Jika masih perlu bantuan, anda bisa kontak kami di:

    5. support@greenhcm.com



English Language
  1. Login Problem or Can't login
    • Check Username / Password / Company Code is Correct?
    • Check Check whether GHCM apps is updated to newest version? if not, you can update it in Google Playstore here & IOS here.
    • Check GPS and mobile signal is On and ok?
    • Check whether Phone permission settings for storage/camera/location already cleared for GreenHCM app?
    • Check whether Phone resources (Storage & Memory) is enough. If not, generally can be resolve by clearing up storage space by deleting unused files and clearing up Phone Memory (RAM)
  2. GreenHCM app won't install
    • Make sure that your phone Android O/S is Android v.5 (Lollypop) and above.
    • Check your phone system Resource. If full, Free up your storage by deleting unused files or uninstall some unused apps, also clear up Phone memory.
  3. You forgot your Username / Password / Company name?
    • For Password, you can reset it from mobile or web by clicking the forgot password link and our system will send you the reset link. For Company code or Username, you can check from the first registration E-mail or SMS Account verification notification that we last sent you.
    • If you still can't find or remember it than you can ask your company's Admin for GreenHCM system to help you by searching it in the system.
    • For last resort, you can contact our customer service to help you.
  4. If you still need help, you can contact us at:

    5. support@greenhcm.com
