Penggunaan Aplikasi GreenHCM

 

1. User Outlet mendapatkan notifikasi dari GreenHCM tentang pengaktifan akun dan informasi akun Login Platform GreenHCM melalui Email / WA.

 

2. User Outlet mendownload Aplikasi GreenHCM:
Untuk handphone Android di Play store: https://play.google.com/store/apps/details?id=com.greenhcm.android
Untuk Handphone Apple di Apple Store: https://apps.apple.com/id/app/greenhcm/id1498577411

 

3. User Outlet melakukan login ke aplikasi mobile GreenHCM menggunakan informasi Login yang dikirimkan sebelumnya via WA/Email. Mohon cek kembali isi Email atau pesan WA dari GreenHCM.
Masukan 'Username', 'Password' dan 'Company Code' yang telah diberikan. Klik:"Log In"

 

4. Setelah login, User Outlet melakukan Check-In di menu 'Daily Journey' dalam aplikasi GreenHCM di lokasi Toko/Outlet untuk menginisialisasi lokasi Toko/Outlet ke dalam sistem GreenHCM untuk pertama kalinya. Pastikan User berada di Outlet/Toko saat melakukan langkah ini.

 

5. User outlet setelahnya bisa melakukan clock-In/Out dengan sistem GreenHCM secara berkala.

 

Terimakasih untuk perhatiannya.
